
1. 변수
	단 하나의 값을 저장할 수 있는 메모리 공간.
	자바에서 메모리 관리는 가비지 컬렉터가 담당하게 됨으로,
	개발자는 메모리 관리를 직접하지 않아도 되어서, 편리함.
	
	변수에는 단 하나의 값만 저장할 수 있음.
	여러개의 값을 저장해서 사용하고 싶은 경우는 배열을 사용해야 함.
	
	- 변수의 선언과 초기화
	int age; //age 라는 변수에 정수형의 데이터를 저장해서 사용하겠다는 의미.
	변수타입에 맞는 킈의 저장공간이 생성. 이 공간 사용하기 위해서 age 공간 이름을
	붙여서 사용하게 됨.
	
	int age = 20;
	변수 age를 선언하고 할당된 그 공간에 정수형태의 값으로 20을 저장.
	
	int a;
	int b;
	
	int a, b;
	변수의 초기화는 변수를 사용하기 전에 처음으로 값을 저장하는 것.
	
	변수 명명 규칙
	- 대소문자 구분. 길이에 제한은 없음.
	- 예약어는 사용 금지.
	- 숫자로 시작해서는 안됨.
	- 특수문자는 '_', '$' 만 가능.
	
	클래스 명명 규칙
	- 클래스 이름의 첫 글자는 항상 대문자이어야 함.
	- 여러 단어로 이루이진 경우는 각 단어의 첫 글자는 대문자로 함.
	- 상수의 이름은 모두 대문자로 함.
	  여러 단어로 이루어진 경우는 '_' 로 구분함.
